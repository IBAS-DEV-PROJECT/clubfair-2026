# 🕹️ CLUB_FAIR_2026

> **2026 동아리 박람회: 실시간 성향 매칭 및 이벤트 관리 플랫폼**  
> 90년대 레트로 OS 감성의 UI와 클라우드 기반 인프라를 통해 전년 대비 방문자 수 **91% 성장**을 기록했습니다.

## 🚀 프로젝트 목적

1. **2026 동아리 박람회 운영 최적화**: 방문객 대상 실시간 매칭 서비스 및 이벤트 참여 기회 제공
2. **기술적 부채 해결 및 인프라 현대화**:
   - **과거 (2025)**: 로컬 노트북 운영으로 인한 시스템 불안정 및 협업/데이터 관리의 한계
   - **현재 (2026)**: 클라우드 기반 SaaS 아키텍처 도입으로 무중단 서비스 실현 및 운영 효율 극대화

## 🛠 Tech Stack

### **Frontend**

- **Core**: React, TypeScript
- **State & Data**: TanStack Query, Zustand
- **Styling**: Styled-components, **React95** (Retro UI)

### **Backend & Infrastructure**

- **Platform**: **Supabase**
  - Database: PostgreSQL
  - Logic: Edge Functions, SQL Functions (RPC)
  - Security: Row Level Security (RLS)
- **Deployment**: **Vercel**
- **Auth**: Supabase Auth

## 📈 프로젝트 성과

로컬 환경의 한계를 클라우드로 전환하며 서비스 안정성을 확보한 결과, 모든 지표에서 압도적인 성장을 기록했습니다.

| 지표               | 2025년 (과거)           | **2026년 (현재)**          | 성장률         |
| :----------------- | :---------------------- | :------------------------- | :------------- |
| **전체 방문자 수** | 약 300명                | **572명**                  | **약 91% ⬆️**  |
| **운영 환경**      | 로컬 노트북 (단일 기기) | 클라우드 (Vercel/Supabase) | 가용성 극대화  |
| **데이터 관리**    | 수동 및 휘발성 관리     | 실시간 대시보드 모니터링   | 운영 효율 증대 |

- **실시간 사용자 몰입도**: 테스트 참여율 **95%** 달성
- **참여 최적화**: 성비(52:48), 게임 참여율(38%) 등 주요 지표 실시간 추적 성공

## 🔑 주요 기술적 해결 과제

### 1. 동시성 제어 및 데이터 무결성 보장

- 다수의 유저가 동시에 이벤트에 응모할 때 발생하는 데이터 충돌 및 부정 차감 문제를 방지하기 위해 PostgreSQL의 **`FOR UPDATE`** 구문을 활용한 트랜잭션 잠금(Row Lock) 로직을 구현했습니다.

### 2. 도메인 분리 및 보안 아키텍처 설계

- **User/Admin 도메인 분리**: 일반 서비스 환경과 운영진 관리 페이지를 물리적으로 분리하여 보안성을 강화했습니다.
- **권한 제어**: 데이터베이스 계층에 **RLS(Row Level Security)** 정책을 적용하고, **`SECURITY DEFINER`** 함수를 통해 민감한 비즈니스 로직(재화 차감, 당첨자 선정 등)을 안전하게 처리했습니다.
